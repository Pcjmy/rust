## README

### 安装rust

https://www.rust-lang.org/zh-CN/tools/install

验证是否安装成功

```shell
rustc --version
```

### 项目初始化

```shell
cargo new hello
```

### 项目运行

```shell
cd hello
cargo build
cd target/debug
./hello
或
cd hello
cargo run
```
